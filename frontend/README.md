# Фронтенд образовательной платформы

## Требования

- Node.js (версия 14.x или выше)
- npm (обычно устанавливается вместе с Node.js)

## Установка зависимостей

Перед первым запуском необходимо установить все зависимости проекта:

```bash
npm install
```

## Запуск приложения

### Режим разработки

Для запуска приложения в режиме разработки выполните:

```bash
npm start
```

Приложение запустится на [http://localhost:3000](http://localhost:3000) и автоматически откроется в браузере.

Страница будет автоматически перезагружаться при внесении изменений в код. Ошибки lint будут отображаться в консоли.

### Сборка для продакшена

Для создания оптимизированной сборки для продакшена выполните:

```bash
npm run build
```

Оптимизированные файлы будут созданы в папке `build/`.

### Запуск тестов

Для запуска тестов в интерактивном режиме выполните:

```bash
npm test
```

## Структура проекта

```
frontend/
├── public/          # Статические файлы
│   └── index.html   # Главный HTML файл
├── src/
│   ├── api/         # API и mock данные
│   ├── pages/       # Страницы приложения
│   ├── App.js       # Главный компонент приложения
│   └── index.js     # Точка входа
├── package.json     # Зависимости и скрипты
└── README.md        # Этот файл
```
